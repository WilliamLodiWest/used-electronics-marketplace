Used Electronics Marketplace

A responsive web marketplace developed as the Final Course Project for the Analysis and Systems Development degree.

The platform allows users to buy and sell second-hand electronic devices through a modern, intuitive and user-friendly interface.

---

## 📖 About the Project

This project was created to provide a simple and efficient marketplace where users can advertise, browse and purchase used electronic devices.

The main objective was to develop a responsive and organized web application focused on usability and user experience.

---

## 🚀 Features

- User registration
- User login
- Product listing
- Product details page
- Search functionality
- Responsive design
- Clean and intuitive interface

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
- MySQL
